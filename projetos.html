Agenda de aniversariantes

FATEC FRANCA (ADS)

Desenvolvi este projeto na aula de Estrutura de Dados do 3º semestre da faculdade. O objetivo é criar uma agenda onde seja possível adicionar pessoas e suas datas de aniversário. Além disso, o sistema permite excluir registros, alterar o dia e o mês de um aniversário, consultar aniversários por data ou mês, buscar por letra inicial, visualizar a agenda organizada por nome ou por mês e, por fim, encerrar o programa.

  • CSS
  • JavaScript
